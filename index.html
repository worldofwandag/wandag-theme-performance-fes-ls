🧪 CSS Variables (Custom Properties)

Store reusable values in your laboratory and change them in one place!

What are CSS Variables?

CSS Variables allow you to:

  • 🧪 Store values (colors, sizes, fonts) in one place
  • 🔮 Reuse them throughout your CSS
  • ⚗️ Change the whole site by editing one line
  • 🎭 Create themes easily
/* Define variables in :root */
:root {
  --primary-color: #6b4c9a;
  --text-color: #1a1a2e;
  --spacing: 16px;
}

/* Use them with var() */
.button {
  background: var(--primary-color);
  color: var(--text-color);
  padding: var(--spacing);
}

🌙 Dark Mode

Create light and dark themes by overriding variables!

How Dark Mode Works

The Pattern:

  1. Define default (light) variables in :root
  2. Create a .dark-theme class that overrides them
  3. Toggle the class on <body> with JavaScript
🦇 Pro tip: You can also use prefers-color-scheme to auto-detect user preference! 
/* Light mode (default) */
:root {
  --background: #f5f5f5;
  --text: #1a1a1a;
}

/* Dark mode override */
.dark-theme {
  --background: #0d0d0d;
  --text: #e8e8e8;
}

/* Elements use variables */
body {
  background: var(--background);
  color: var(--text);
}
🌓 Dark Mode Toggle
/* Current mode: Dark */

.dark-theme {
  --bg-color: #0d0d0d;
  --text-color: #e8e8e8;
  --card-bg: #1a1a2e;
  --border-color: #333;
}
/* JavaScript toggle */
document.body.classList.toggle('dark-theme');

🐌 Bad vs Good Selectors 🚀

❌ Bad (Deep Nesting)

body div section ul li a {
  color: red;
}
  • 🐌 Slow to match
  • 😵 Hard to read
  • 🔗 Tightly coupled to HTML
  • 😰 Breaks if HTML changes

✅ Good (Flat Class)

.nav-link {
  color: red;
}
  • 🚀 Fast to match
  • 👀 Easy to read
  • 🔓 Flexible, reusable
  • 💪 Robust to changes

📜 DRY: Don't Repeat Yourself

❌ Repetitive

.card1 { background: #1a1a2e; }
.card2 { background: #1a1a2e; }
.card3 { background: #1a1a2e; }
.card4 { background: #1a1a2e; }
.card5 { background: #1a1a2e; }

😫 If you need to change the color, you edit 5 places!

✅ Reusable

.card {
  background: var(--card-bg);
}

🎉 One class, uses a variable. Change once, updates everywhere!

✨ Best Practices Summary

🎯

Use flat selectors

Prefer .class over deep nesting

🔄

Reuse classes

Create utility classes you can combine

🎨

Use CSS variables

For colors, spacing, and theme values

📁

Keep CSS organized

Group related styles together

🎬

Animate wisely

Prefer transform and opacity

🧹

Remove unused CSS

Less code = faster load times
